### ops/RECOVERY-AND-MIGRATION-GUIDE.md

**Issues: 2 (1 security)**

1. **[SECURITY] Live credentials in plaintext.** Step 4 (Restore API keys) shows:
   ```
   ARGUSREACH_BOT_TOKEN=  ← 8588914878:AAEQnZNXWx9_j2llD-Yw0sWwjegXu-pruCk
   ARGUSREACH_CHAT_ID=    ← 8135725412
   ```
   The actual Telegram bot token and chat ID are hardcoded in the document. If this file is ever saved to Google Drive, shared, or accidentally leaked, the bot could be compromised. Credentials should use placeholders (e.g., `← from BotFather settings`) not real values.

2. **[MISSING] Prospect CSV restoration.** The recovery guide covers restoring code, API keys, and DNC lists but doesn't mention restoring the campaign prospect CSVs (e.g., `campaigns/pt-tampa-bay-test/prospects.csv`). If this file is missing after a rebuild, the prospect filter won't work.

**Changes needed:**
- Replace real Telegram bot token and chat ID with placeholders and instructions to retrieve them
- Add prospects.csv to the list of things to restore in Part 2
