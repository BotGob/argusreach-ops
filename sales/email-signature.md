## How to set up in Gmail (vito@argusreach.com)

1. Gmail → Settings (gear icon) → See all settings
2. General tab → Signature → Create new
3. Name it "ArgusReach Standard"
4. Paste the Standard version above
5. Set as default for new emails
6. Create a second signature named "ArgusReach Extended" with the extended version
7. Use extended when cold prospecting; standard for ongoing threads
