## First Client Acquisition

### Step 1: Build Your First Prospect List (this week)
1. Log into Apollo.io
2. Search: Title = "Founder" OR "Managing Partner" OR "Owner", Industry = "Financial Services", Location = Tampa Bay FL, Employees 2–50
3. Export 20–30 names
4. Add to Airtable manually

### Step 2: Send First Outreach (Vito himself — on behalf of ArgusReach)
Use the RIA sequence from `email-sequences-v1.md` — adapt it so Vito is pitching ArgusReach's services to the RIA directly (not outreaching on behalf of an RIA).

Subject line idea:
`[Firm] — building a proactive pipeline outside of referrals?`

### Step 3: Strategy Call
- Run 15-min call using `pitch-deck-script.md`
- Send proposal same day using `proposal-template.md`
- Collect $500 setup fee via invoice (Stripe or Venmo Business)
- Schedule onboarding call

### Step 4: Run First Campaign
- Follow `sop-campaign-launch.md`
- Use Instantly.ai warmed domain
- Report after Week 1
